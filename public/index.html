📚 Manga Affiliate Manager

Connect to WordPress

The tool will fetch all your manga categories and let you add an affiliate link to each one's description.

Generate an Application Password in WordPress: Dashboard → Users → Your Profile → Application Passwords.
The REST API must be enabled (it is by default on WordPress 5.0+).

Match & Review

Upload your CSV to match Amazon links to manga categories, then approve each one.

Example: One Piece,https://amzn.to/xxxxxx
Titles are matched case-insensitively. Partial matches work too.

Publish to WordPress

The affiliate section will be appended to each approved manga category's description.

Clicking Run publish will update each approved category via the WordPress REST API. Categories that already have an affiliate section will be skipped automatically — no duplicates.
